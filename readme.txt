# Recruitment System

The Recruitment System is a web application designed to streamline the recruitment process for organizations.
It provides features for managing job postings, applications.

## Features

- Job posting management: Create, edit, and delete job postings with details such as job title, description, qualifications, and application deadline.
- Application tracking: Receive and manage job applications, including candidate information.
- Self-registration for both user types-  ( Employer and Applicant )
